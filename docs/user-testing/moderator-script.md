# Moderator Script

Read naturally; adapt wording but keep task order. Do not show later tasks in advance.

## Welcome

> Thanks for helping test the GeoSpatial Data Quality Agent. We are evaluating the website, not your skills. Please think aloud as you work. I will give you one task at a time. You can stop anytime.

Hand [participant-brief.md](participant-brief.md). Obtain signed or verbal consent per [consent-template.md](consent-template.md).

> The test file is `sample.geojson` on your machine at: *[give path to datasets/examples/sample.geojson]*.

---

## T1 — Upload

> Please upload the sample GeoJSON file using the Upload page. Tell me when you believe the dataset is loaded.

**If stuck:** "Which navigation tab might relate to bringing data in?"

**If still stuck (>2 min):** Demo Upload control once; mark assistance `demo`.

---

## T2 — Validate

> Open the Dashboard and run validation on your dataset. Let me know when you see validation results.

**If stuck:** "Look for a button that starts validation."

---

## T3 — Explore issue

> Find a critical-severity issue, select it, and read the details and any suggested fix in the detail panel.

**If stuck:** "Try the filters above the issues list, or click a marker on the map."

---

## T4 — Decide corrections

> For two different issues that have suggestions, approve one correction and reject another. Tell me when both choices are visible in the list.

**If stuck:** "Open an issue in the detail panel—Approve and Reject should be there."

---

## T5 — Apply corrections

> Apply your correction choices using the button on the Dashboard. Tell me when the application confirms the result.

**If Apply is disabled:** Ask if any Custom issues need saving; check pending custom message.

---

## T6 — Report

> Open the quality report, skim the summary, and export or print the report in any format offered.

**If stuck:** "Use the Report link in the top navigation, or View report on the Dashboard."

---

## SUS

> Please answer these ten questions about the system overall, based on your experience today. There are no right answers.

Hand [sus-questionnaire.md](sus-questionnaire.md) or digital form. Collect 1–5 per item.

---

## Debrief

1. What felt most confusing?
2. What worked well?
3. Would you use this tool for real data cleaning? Why or why not?
4. Anything we did not ask you to try that you wanted to do?

Thank participant; explain data retention (anonymous IDs only in research notes).
